SIZE

width

Utilities for controlling the aspect ratio of an element.

Attribute Style
w="full" width: 100%
w="screen" width: 100vw
w="fit" width: fit-content
w=<number> width: calc(var(--spacing) * <number>)
w=[<value>] width: <value>
max-w=<number> max-width: calc(var(--spacing) * <number>)
max-w=[<value>] max-width: <value>
min-w=<number> min-width: calc(var(--spacing) * <number>)
min-w=[<value>] min-width: <value>
size="full" width: 100%
height: 100%
size="screen" width: 100vw
height: 100vh
size="fit" width: fit-content
height: fit-content
size=<number> width: calc(var(--spacing) * <number>)
height: calc(var(--spacing) * <number>)
size=[<value>] width: <value>
height: <value>
